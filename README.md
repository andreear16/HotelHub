# HotelHub

Aplicație web realizată în cadrul disciplinei **Dezvoltarea Aplicațiilor Web (DAW)**.

**HotelHub** este o aplicație web destinată administrării activităților unui hotel modern – de la rezervări și evidența clienților, până la facturare și gestionarea personalului.

## Structură
- `index.html` – Introducere și scopul aplicației  
- `descriere.html` – Descriere generală și diagrame UML  
- `baza_date.html` – Modelul conceptual al bazei de date  
- `css/style.css` – Stiluri  
- `images/` – Imagini UML și ERD
