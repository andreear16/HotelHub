# 🏨 HotelHub

**HotelHub** este o aplicație web realizată în cadrul disciplinei **Dezvoltarea Aplicațiilor Web (DAW)**, având ca scop digitalizarea și simplificarea administrării unui hotel modern.

Aplicația oferă o interfață clară și intuitivă, care permite gestionarea rezervărilor, camerelor, clienților și personalului într-un singur loc.

## 🌐 Demo online

🔗 [Accesează aplicația HotelHub aici](https://andreear16.github.io/HotelHub/)

---

## 📘 Descriere generală

Proiectul este structurat pe mai multe pagini web care descriu funcționalitatea și arhitectura sistemului:

- **`index.html`** – Introducerea și scopul aplicației  
- **`descriere.html`** – Descriere generală și diagrame UML  
- **`baza_date.html`** – Modelul conceptual al bazei de date și relațiile dintre entități  

Aplicația este dezvoltată în **HTML**, **CSS** și **PHP/MySQL** (pentru partea logică), urmărind principiile unui sistem real de administrare hotelieră.

## 🧩 Funcționalități principale

- Gestionarea **rezervărilor** și **camerelor** disponibile  
- Evidența **clienților** și a **angajaților (recepționeri)**  
- Emiterea **facturilor** pentru rezervări confirmate  
- Administrarea **serviciilor suplimentare** oferite clienților  
- Model de bază de date cu relații între entități (1:1, 1:N, M:N)
